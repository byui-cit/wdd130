  4. Make the Grid more responsive

    Your page should now be looking finished and good...as long as the browser window is wide enough. Try reducing the width of your browser however. What do you notice?

    At narrow widths the image gets really small, and the text really long. Not ideal. It would be good to change the layout slightly when the screen gets too small. One way we can do this is by taking advantage of CSS grid repeat() and auto-fit. Review again the article on auto sizing columns in grid that you should have read earlier this week.

    Based off of the contents of that article see if you can modify your grid-template-columns to make it so that the columns can never be smaller than 400px, and never bigger than 1fr. Use repeat and auto-fit as well so that our grid will switch between one and two columns automatically as needed. Once you have your solution you can check the solution below

    Auto-fit grid columns 
    .photo {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(400px, 1fr));
    gap: 2em;
}
